Documentation - Levels

The levels configuration section is used to define the color scheme used by the Harvester console as well as define any aliases for each log level.

<levels timestampColor="Cyan">
    <fatal backColor="Red"   foreColor="White"    aliases="fatal" />
    <error backColor="Black" foreColor="Red"      aliases="error" />
    <warn  backColor="Black" foreColor="Yellow"   aliases="warn,warning" />
    <info  backColor="Black" foreColor="White"    aliases="info,information" />
    <debug backColor="Black" foreColor="DarkGray" aliases="debug" />
    <trace backColor="Black" foreColor="DarkCyan" aliases="trace" />
</levels>

aliases

A comma (,) or semi-colon (;) delimited list of string values that should map to the associated log level.
Default Value: Corresponding SystemEventLevel (i.e., debug).
Applies To: Console, Windows

backColor

A Console Color used to highlight the text component of a message.
Default Value: ConsoleColor.Black.
Applies To: Console Only

foreColor

A Console Color used to render the text component of a message.
Default Value: ConsoleColor.DarkGray.
Applies To: Console Only

timestampColor

A Console Color used to render the timestamp component of a message.
Default Value: ConsoleColor.Cyan.
Applies To: Console Only
